# 探せ - サポートページ

「探せ」は、画面のどこかに隠れている人物を見つけ出すシンプルな探し物ゲームです。

## 遊び方

- スタートすると、画面のランダムな位置に人物が配置されます
- タップして正しい人物を見つけ出しましょう
- 毎回配置が変わるので、何度でも新鮮に遊べます

## お問い合わせ・不具合報告

ご意見・不具合報告は、このリポジトリの [Issues](../../issues) からお願いします。

または以下のメールアドレスまでご連絡ください。
- your-email@example.com

## プライバシーポリシー

本アプリは、ユーザーの個人情報を収集・保存・送信しません。
すべてのゲームデータは端末内に保存され、外部サーバーへ送信されることはありません。
